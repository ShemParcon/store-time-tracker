How to connect Google Sheets
1

Go to sheets.google.com and create a new spreadsheet. Add headers in row 1: Timestamp, Name, Type, Date, Time

2

Click Extensions → Apps Script. Delete any existing code and paste the script provided in the README (or ask for it).

3

Click Deploy → New deployment → Web app. Set "Who has access" to Anyone. Copy the Web App URL.

4

Paste that URL below and tap Save.
